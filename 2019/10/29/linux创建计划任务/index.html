linux计划任务
发表于 2019-10-29| 更新于 2019-11-14|linux定时任务

crontab -r #删除所有任务
crontab -l #列出当前的定时任务
crontab -e #编辑crontab内容

第一次使用如果出现如下提示，是选择编辑器

选择vim编辑器

重新选择编辑器
select-editor

执行crontab -e

redhat查看日志

Ubuntu 需要手动开启 Crontab 计划任务日志
